C程序数据类型

咱们可以把 C语言的数据类型 想象成 不同规格的储物容器，用来存放各种数据。就像你存水用水杯、存书用书包一样，不同的数据类型对应不同的存储需求和大小

📦 数据类型——数据的「房子」

作用：决定数据在内存中占多大空间，能表示什么范围的数值。

🗂️ 四大类「房子户型」

  1. 基本类型（单身公寓）

    • 整型：存整数 → int（标准间）、short（小单间）、long（大平层）
    • 字符型：存单个字符 → char（迷你仓，其实也是小整数）
    • 浮点型：存小数 → float（普通水杯）、double（大桶水）
    • 例子：

  2. 枚举类型（VIP专属房）

    • 只能存放预设的整数值（比如星期几、方向）
    • 例子：

  3. void类型（隐形房）

    • 三副面孔：
      • 函数不返回值 → void 吃饭() { ... }
      • 函数不需要参数 → int 随机数(void) { ... }
      • 万能指针（像万能钥匙，但要指定类型后才能用） → void* 神秘地址;

  4. 派生类型（组合别墅）

    • 后续章节解锁：数组（一排连号房）、指针（门牌号）、结构体（套房）

🔢 整型「房型对比表」

类型 存储空间 值范围（64位系统） 生活类比
char 1字节 -128127 或 0255 小抽屉（存年龄）
unsigned int 4字节 0~4,294,967,295 超大储物箱
long 8字节 -9万亿~9万亿 仓库

📢 关键技巧

  • 选类型就像选背包：够用就行，别浪费空间

🌊 浮点型「精度段位」

类型 存储空间 精度 适用场景
float 4字节 6~7位小数 日常测量（如体重）
double 8字节 15~16位小数 科学计算
long double 16字节 19位小数 超高精度需求

💡 类型转换口诀

显式转换需手动，砍头去尾要谨慎。
浮点转整丢小数，大数化小会失真。
混用类型先统一，避免暗坑代码稳。

附：测试你理解的灵魂三问 🧐

  1. (int)3.99等于多少？ → 3（直接砍掉小数）
  2. char c = 1.5 + 2.7;中c的值？ → 4（隐式转int=4，再转char）
  3. int a = 5; double b = a/2;b的值？ → 2.0（先做整数除法5/2=2，再转double）
    修正方法：double b = (double)a/2; → 得到2.5
