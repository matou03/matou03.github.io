三、define vs const 大比拼 🥊

特性 #define ✨ const 🔒
类型检查 ❌ 无 ✅ 有
作用域 全局 局部/全局
内存占用 无（文本替换） 有
调试 难（替换后看不到原名） 易（保留变量名）

总结：

  • 推荐多用const（更安全、易调试）
  • #define适合简单替换或跨文件常量
